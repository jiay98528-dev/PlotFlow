# 证据包 3：键盘与辅助技术

从已保存的有效故事开始，不修改产品配置：

- 纯键盘验证章节 Tabs Left/Right/Home/End，节点菜单 ContextMenu/Shift+F10、方向键、Enter/Space/Escape。
- 验证 Ctrl+K 搜索、结果 live region、方向键、Escape，以及焦点回到触发器、目标节点或 Inspector。
- 用键盘开关 Palette、Inspector、Source Drawer、诊断和 ConditionEditor。
- 验证 ConditionEditor dialog、初始焦点、Tab 循环、操作数名称和 AND/OR/NOT pressed 状态。
- 开启 Narrator，抽查主要命令、图标按钮、章节、节点和诊断，不得出现无名称按钮或语言混杂。

停止点：Narrator/录屏不可用为 `BLOCKED`；可稳定复现的无名称、焦点丢失、键盘陷阱或不可操作控件为 `FAIL`。
