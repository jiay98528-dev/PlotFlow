# 第一章：村庄

## 节点：森林入口

你站在幽暗森林的边缘，
两条小径延伸向前。

[选项] 走向左边的狼嚎声 -> 节点：狼穴
  效果: (好感度+1)

[选项] 投喂食物 -> 节点：驯服狼
  条件: ($金币>=10) AND ($武器!='无')
  效果: (金币-10, 好感度+5)

Markdown 写作，零学习成本

用 ## 节点： 定义剧情节点，[选项] 创建分支。 语法就是 Markdown——任何文本编辑器都能打开，Git diff 友好，永不锁死数据。

实时分支可视化

React Flow 驱动的节点图，自上而下树状布局。 拖拽连线即可修改跳转目标——图形和文本始终保持双向同步。

多格式导出，引擎就绪

一键导出标准 JSON（含条件 AST）+ 可玩 HTML + 纯文本 TXT。 Godot 编辑器插件 + 运行时库完整交付，Unity/Unreal 接口预留。

Monaco 编辑器

VSCode 同款内核。PlotFlow 语法高亮、节点折叠、大纲导航、括号自动闭合。暗色/亮色主题一键切换，中英双语完整覆盖。

Monarch Tokenizer

可编辑分支图

React Flow + Dagre 自上而下层级布局。节点卡片显示标题+摘要+选项数。条件连线虚线，无条件连线实线。拖拽连线端点即可修改跳转目标，文本自动同步。

React Flow · Dagre

图形化条件编辑器

Airtable 过滤条件构建器风格。变量下拉 + 比较运算符 + 值输入，AND/OR 逻辑组合最多嵌套 3 层。修改面板自动同步文本，反之亦然。

零代码 · 双向同步

四维幽灵补全

节点标题 / 选项句式 / 正文描述 / 变量名——四个维度独立触发。N-gram 统计模型纯本地运行，用户保存时增量学习写作风格。灰色幽灵字符静默提示，Tab 接受，Esc 忽略。

离线 N-gram · 隐私安全

三级错误检测

8 种红色错误（阻断导出）+ 6 种黄色警告 + 3 种蓝色建议。Monaco 波浪线 + 侧边标记 + 悬停 tooltip + 问题面板，每条诊断附带可操作修复建议。
