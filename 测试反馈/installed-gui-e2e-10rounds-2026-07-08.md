## 已确认阻断项

### BUG-GUI-001 原生保存对话框标题乱码

严重级别：P0

发生位置：首次保存 / Save As 原生 Windows 对话框。

复现轮次：第 1 轮。

实际表现：窗口标题显示为 `淇濆瓨 PlotFlow 鏁呬簨鏂囦欢`，不是正常中文“保存 PlotFlow 故事文件”。

影响：发行包本地化质量不达标；用户在保存核心闭环中直接看到乱码。

### BUG-GUI-002 原生保存对话框可能落到下层，主界面卡在 saving 状态

严重级别：P0

发生位置：首次保存 / Save As 原生 Windows 对话框。

复现轮次：第 1 轮。

实际表现：触发保存后，主窗口底部持续显示 `saving⏳ 正在打开保存对话框...`。窗口列表中仍存在隐藏/下层保存窗口，需要手动激活该窗口后取消，主界面才恢复。

影响：用户可能认为软件卡死，保存闭环无法可靠完成。

### BUG-GUI-003 保存异常后 Web 文档出现大面积文本选择高亮

严重级别：P1

发生位置：取消隐藏保存窗口后回到主界面。

复现轮次：第 1-2 轮。

实际表现：Graph Lab、Theme Center、Inspector 等大面积 UI 文本处于蓝色选中状态，污染视觉状态。

影响：视觉体验明显异常；后续操作虽可继续，但界面状态噪声高。
